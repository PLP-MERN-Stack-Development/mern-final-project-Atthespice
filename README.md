# Community Food Bridge

A platform connecting food donors with recipients to combat food waste and support communities in need.

## About

Community Food Bridge is a web application designed to facilitate food sharing between donors (restaurants, grocery stores, caterers, individuals) and recipients (food banks, shelters, community centers, individuals). The platform aims to reduce food waste while helping those who need nutritious meals.

## Features

- **Role-Based Authentication**: Separate flows for food donors and recipients
- **User Dashboard**: Personalized interface based on user role
- **Responsive Design**: Beautiful, mobile-friendly interface with warm community-focused design


## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom design system
- **UI Components**: shadcn-ui
- **Form Handling**: React Hook Form with Zod validation


## Installation

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn package manager
